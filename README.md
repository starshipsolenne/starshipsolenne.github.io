# Solenne-LCARS-Website

## Custom Components

### crewlist

An unordered list (UL) used to display crew manifest. Position (DIV) and link (A) are optional, if not provided then they do not appear.

```html
<ul class="crewlist">
    <li>[Name]
        <div>[Position]</div>
        <a href="[Link]"></a>
    </li>
</ul>
 ```

### submenu

A menubar for use within a page.

```html
<div class="submenu">
    <a href="[Url]">[Link 1]</a>
    <a href="[Url]">[Link ... n]</a>
    <div>&nbsp;</div>
</div>
```
